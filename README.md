# Калькулятор стоимости авто «под ключ»

CLI-инструмент для расчёта итоговой цены автомобиля с учётом таможни, логистики и комиссии.

## Формула расчёта

- Таможня: 48% от стоимости авто
- Логистика: 150 000 ₽ (фиксированная)
- Комиссия: 7% от (цена + таможня + логистика)

## Установка и запуск

git clone https://github.com/motok0t/car-price-calculator.git

cd car-price-calculator

python main.py

## Поддерживаемые форматы ввода цены

- 1500000
- 1,500,000
- 1.500.000
- 1 500 000
- 1_500_000

## Пример работы

Марка и модель: BMW X5

Стоимость: 7 000 000 ₽

ИТОГО «ПОД КЛЮЧ»: 11 245 700 ₽

## Категории авто (определяются по ключевым словам в названии)

- Премиум: BMW, Mercedes, Audi, Lexus, Porsche
- Экономичные: Kia, Hyundai, Lada, Renault
- Стандартные: все остальные марки (Chery, Tesla, Toyota, Ford и др.)

*Категория влияет только на отображение, не на формулу расчёта.*
